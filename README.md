# Rocket Media

Projet sous [Angular CLI](https://github.com/angular/angular-cli) en version 14.0.1. Ce projet a pour but de s'exercer et progresser avec Angular.

## Installation

Cloner le projet
```
ng clone https://github.com/behsse/rocket-media.git
```

Installer le projet
```
npm i
```

Lancer le serveur
```
ng serve -o
```
